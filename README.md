# 🎬 BookMyShow Backend (Spring Boot)

## 🚀 Features
- User authentication
- Movie listing
- Show scheduling
- Seat booking system
- Payment integration logic

## 🛠️ Tech Stack
- Java
- Spring Boot
- MySQL
- JPA / Hibernate

## 🔗 APIs
- GET /api/movies
- GET /api/shows
- POST /api/bookings

## 💡 Highlights
- Clean layered architecture (Controller → Service → Repository)
- Exception handling implemented
- Scalable design

## ▶️ Run Locally
mvn spring-boot:run
